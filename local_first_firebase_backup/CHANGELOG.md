## 0.1.0

* Initial release
* Firebase Storage backup provider for the LocalFirst framework
* Cross-platform support (Android, iOS, macOS, web, Windows, Linux)
* Per-user backup isolation via Firebase Auth UID
* Upload, download, list, and delete backup operations
* Configurable subfolder within user's backup directory
* Lazy Firebase instance resolution for testability
* 100% test coverage with mocked Firebase APIs
